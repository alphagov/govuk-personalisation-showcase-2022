Objectives

  1. Explore concept of connecting external data; are users open and willing to:
    • pull from external data source, and/or;
    • push from GOV.UK account to external service
  2. Explore concept of automating recurring transactions
  3. Explore willingness to connect payment options to GOV.UK account
  4. Check comprehension of audit trail and assess perceived value of this within the context of the GOV.UK account dashboard

Hypotheses

External data


we believe that linking external data

will result in result in some nervousness and resistance

because

 

we believe that linking external data as part of a recurring transaction

will result in a greater willingness to engage

because there is value in time saving through recurring transactions

 

we believe that pushing to external data sources

will be dependent on what the thing is

because users are willing to share some data items rather than others

 

we believe that pushing to external data sources

will result in greater time saving and efficiency

because users do not have to spend time retrieving information and / or setting up transactions manually




Automated recurring transactions


we believe that automating recurring transactions

will be perceived as valuable

because users are saved time in entering the same details repeatedly




Payment details


we believe that saving payment details to account

will result in greater time saving

because users will not have to enter the same details repeatedly




Audit trail


we believe that recording transactions

will result in greater confidence

because users have a point of reference if and when they need it

 

we believe that recording transactions

will result in greater time saving and lesser cognitive load

because users will not have to take responsibility to keep their own records
